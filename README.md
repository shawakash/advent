# 🎄 Advent of Code Solutions

This repository contains my solutions to [Advent of Code](https://adventofcode.com/) challenges implemented in multiple programming languages.

## 📚 Structure

```
advent-of-code/
├── 2023/
│   ├── rust/
│   │   └── ...
│   └── go/
│       └── ...
└── 2022/
    └── ...
```

## 🚀 Languages

Solutions are implemented in:
- Rust 🦀
- Go 🔵

## 💻 Running Solutions

### Rust
```bash
cd 2023/rust/
cargo run
```

### Go
```bash
cd 2023/go/
go run main.go
```
